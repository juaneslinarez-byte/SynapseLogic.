# SynapseLogic - Análisis de Conectividad y Transmisión Neuronal

Simulador de redes neuronales representadas como grafos dirigidos. Permite cargar redes desde archivos CSV, analizar conectividad mediante BFS y DFS, calcular rutas óptimas de transmisión sináptica con el algoritmo de Dijkstra, y simular fatiga cognitiva. La visualización del grafo se realiza con GraphStream.

- Java 25
- Swing (interfaz gráfica)
- GraphStream 2.0 (visualización del grafo)

```

└── synapselogic/

```
